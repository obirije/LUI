Privacy Policy

Last updated: April 2026

Summary

  • LUI stores API keys locally on your device in encrypted storage
  • LUI does NOT collect, transmit, or sell personal data
  • Voice/text queries may be sent to cloud providers you configure
  • LUI does NOT have servers that receive your data
  • All features work locally unless you explicitly enable cloud services

Data Stored on Device

DataPurpose
Chat historyPersistent conversations (Room/SQLite)
API keysEncrypted storage (AES256-GCM)
Notification digestBatched non-urgent notifications
Trigger rulesGeofence and scheduled actions
Downloaded modelOn-device LLM for offline use

None of this data is transmitted to LUI developers or any third party.

Cloud Services (When You Enable Them)

When you configure a cloud LLM (Gemini, Claude, OpenAI, Ollama), your queries are sent to that provider's API. Each provider has its own privacy policy. LUI sanitizes API keys from all data sent to cloud providers.

When you enable cloud speech (Deepgram, ElevenLabs), audio/text is sent to the speech provider.

Web search uses DuckDuckGo. URL browsing uses Jina Reader API.

Accessibility Service

LUI uses AccessibilityService to read screen content, tap buttons, and type text in other apps. This is only active when LUI is set as your default launcher. No data from the accessibility service is transmitted off-device.

BYOS Bridge & Relay

The WebSocket bridge runs locally on your phone. The relay server is a stateless pass-through proxy — no messages are stored. Auth tokens are sent via encrypted WebSocket messages, never in URLs.

Data Deletion

All data is stored locally. Uninstall the app or clear app data in Android Settings to delete everything.

Children

LUI is not directed at children under 13.
